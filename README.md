AIDE
Algoritmo Integrado de Diagnóstico Etiológico para Dislipidemias en Atención Primaria

Herramienta clínica de acceso libre diseñada para ayudar a médicos de atención primaria a evaluar, clasificar y orientar el diagnóstico etiológico de dislipidemias de forma estructurada y rápida.

El proyecto implementa una versión digital interactiva del algoritmo AIDE-SMNE, permitiendo aplicar el proceso diagnóstico directamente desde un navegador web.

Objetivo del proyecto

El objetivo de este proyecto es traducir un algoritmo clínico en una herramienta digital simple, abierta y reproducible que facilite:

Evaluación sistemática de pacientes con dislipidemia
Clasificación fenotípica del perfil lipídico
Identificación de causas secundarias
Sospecha de dislipidemias primarias
Estratificación inicial de riesgo cardiovascular
Orientación sobre cuándo derivar a especialista

La herramienta está pensada principalmente para médicos de primer contacto, pero puede ser útil también para residentes, estudiantes y especialistas.
